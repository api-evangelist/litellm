LiteLLM now integrates with Microsoft ASSERT, an open-source framework for policy-driven agent evaluation built on Microsoft Research methodology. The partnership enables teams to evaluate AI agents against organizational policies across the 100+ language model providers supported by LiteLLM's gateway. Organizations can identify potential safety and quality defects before deploying agents to production environments.
